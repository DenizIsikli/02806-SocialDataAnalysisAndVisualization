# NYC 311 Data Analysis - Complete Project Guide

## Project Overview

This is a comprehensive data analysis project showcasing skills learned throughout the DTU 02806 Social Data Analysis and Visualization course. The project demonstrates:

- **Data Analysis**: Pandas, NumPy operations on real NYC complaint data
- **Statistical Analysis**: Descriptive statistics, equity analysis, trend identification
- **Visualization**: Interactive Plotly charts integrated into web dashboard
- **Data Engineering**: CSV processing, JSON export, data pipeline automation
- **Web Integration**: Dynamic data loading from Python-generated outputs

## Quick Start

### Prerequisites

Ensure you have Python 3.10+ and the required packages:

```bash
pip install pandas numpy plotly
```

### Step 1: Generate Analysis Data

From the `final_project` directory:

```bash
python analysis/generate_data.py
```

**Output**: This generates 6 JSON files in `data/output/` with all visualization data.

### Step 2: View the Dashboard

Open the web dashboard in your browser:

```
final_project/web/index.html
```

The dashboard will automatically load the generated data and display interactive charts.

## What Gets Generated

### Data Pipeline Output

When you run `generate_data.py`, it creates:

| File | Content | Records |
|------|---------|---------|
| `top_complaints.json` | Top 6 complaint categories with counts | 6 |
| `borough_data.json` | Borough statistics (counts, percentages, avg resolution time) | 5 |
| `time_series.json` | Monthly complaint volumes (2020-2025) | 32 |
| `seasonal_data.json` | Complaint counts by season | 4 |
| `equity_data.json` | Average resolution times by borough | 5 |
| `complaint_trends.json` | Complaint trends 2020-2024 with percent change | 6 |

**Location**: `final_project/data/output/`

## How It Works

### Data Flow

```
Raw Data (CSV)
    ↓
[generate_data.py]
    ↓
Statistical Analysis (Pandas operations)
    ↓
JSON Export
    ↓
[index.html - fetch API]
    ↓
Interactive Visualizations (Plotly)
```

### Key Analysis Steps

The `generate_data.py` script:

1. **Loads** processed CSV data from `data/processed/`
2. **Aggregates** complaint data by category, borough, and time
3. **Calculates** statistics (counts, percentages, averages)
4. **Exports** results to JSON for web consumption
5. **Displays** summary statistics and file listing

### Web Integration

The `index.html` dashboard:

1. **Loads** JSON files using Fetch API
2. **Processes** data with JavaScript
3. **Renders** interactive Plotly charts
4. **Responds** to user interactions (hover, zoom, etc.)
5. **Displays** data-driven insights

## Analysis Highlights

### Key Findings

**1. Top Urban Problems**
- Heat: 13.8% of complaints
- Water: 11.3% of complaints  
- Rodent: 10.5% of complaints

**2. Geographic Patterns**
- Manhattan: 25.1% (highest density)
- Brooklyn: 21.9%
- Queens: 20.1%
- Bronx: 20.0%
- Staten Island: 12.9%

**3. Service Equity**
- Average resolution: ~30 days across all boroughs
- Variance: Only 0.6 days
- Conclusion: Fair geographic distribution

**4. Temporal Trends**
- Stable volume 2020-2024 (~800/month)
- Slight increase in major categories (+10-20% by 2024)

## Troubleshooting

### Charts Not Displaying

Check browser console for errors:
- Right-click → Inspect → Console tab
- Verify JSON files exist in `data/output/`
- Check CORS (may need local server if running from file://)

### Data Not Updating

1. Run `python analysis/generate_data.py` again
2. Refresh browser (Ctrl+Shift+R for hard refresh)
3. Check `data/output/` for updated JSON files

### Python Errors

Verify dependencies:
```bash
pip install --upgrade pandas numpy plotly
```
