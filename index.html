Week 7 — Course Summary

Week 1 — Introduction & Data Overview

  • Explored the San Francisco crime dataset and understood its structure
  • Investigated basic properties such as time span, categories, and data quality
  • Created initial plots to understand overall crime trends

  • Analyzed how crime varies over time (years, months, hours)

  • Explored distributions using jitter plots, box plots, and QQ plots
  • Analyzed how crime patterns vary across police districts
  • Found distinct crime profiles in different districts
  • Observed limitations in data precision

Week 4 — Spatial Analysis & Correlation

  • Investigated spatial distribution of crime using grid-based methods
  • Found strong concentration in hotspots
  • Analyzed correlations between crime types
  • Identified shared and distinct temporal patterns

Week 5 — Geographical Visualization

  • Created interactive maps using Plotly and Folium
  • Visualized crime distributions across districts
  • Identified geographic hotspots
  • Learned strengths of map-based visualization

Week 6 — Interactive & Animated Visualization

  • Built interactive charts using Plotly
  • Created animated visualizations over time
  • Observed stable daily crime rhythms
  • Learned benefits of interactivity

Week 7 — Building a Website

  • Learned basic HTML structure
